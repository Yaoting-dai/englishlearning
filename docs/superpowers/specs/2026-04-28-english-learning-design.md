# 幼儿英语启蒙学习 Web App — 设计文档

## 1. 概述

### 1.1 项目目标
为 3-15 岁儿童（幼儿园至初中）设计一款 iPad 浏览器端英语启蒙学习 Web App，涵盖字母、音标、单词、儿歌、美文阅读和 AI 对话六大模块。

### 1.2 目标用户
| 阶段 | 年龄 | 学习重点 |
|------|------|---------|
| 🌱 幼儿园 | 3-6岁 | 字母认读、简单儿歌、基础单词（以听为主，大量图片动画） |
| 📚 小学 | 6-12岁 | 音标、单词跟读、儿歌、简单美文、AI对话（系统学习） |
| 🎯 初中 | 12-15岁 | 进阶单词、深度美文、AI自由对话（阅读量和口语输出） |

### 1.3 平台
- **目标设备**：iPad（Safari 浏览器）
- **Web 标准**：PWA（可添加到 iPad 桌面，全屏体验）
- **响应式**：适配 iPad 横竖屏

---

## 2. 技术栈

| 层级 | 技术方案 |
|------|---------|
| 前端框架 | Next.js 14 (App Router) + TypeScript |
| 样式 | Tailwind CSS |
| 离线能力 | PWA (Service Worker + Manifest) |
| AI 对话 | 火山引擎 ARK (豆包大模型 API) |
| 发音评测 | Azure Speech Pronunciation Assessment |
| 语音合成 (TTS) | Azure Speech (AnaNeural 女童音色) |
| 数据存储 | LocalStorage / IndexedDB（MVP 无需后端） |
| 内容管理 | 静态 JSON / MDX 内嵌 |

### 2.1 语音方案详解
- **评测**：Azure Pronunciation Assessment API，音素级评测，输出准确度/流利度/完整性评分
- **音色**：`en-US-AnaNeural`（Azure 原生女童音色），通过 SSML `prosody pitch/rate` 参数调节适配更低龄用户
- **幼儿园模式**：pitch+15%, rate+10%（更可爱）
- **小学/初中模式**：默认 AnaNeural 参数

## 3. 页面路由结构

```
/                           ← 首页（按年龄推荐内容）
/letters                    ← 字母认读（滑动卡牌）
/phonetics                  ← 音标认读
/words                      ← 单词跟读分类入口
/words/[category]           ← 分类单词学习
/songs                      ← 儿歌列表
/songs/[id]                 ← 儿歌播放器（滚动歌词）
/reading                    ← 美文阅读分类入口
/reading/[id]               ← 文章阅读页
/ai-chat                    ← AI 对话
/settings                   ← 家长设置（幼儿园/小学/初中）
```

## 4. 功能模块设计

### 4.1 首页与导航
- **底部常驻导航**：首页 / 字母 / 学习(单词+音标) / 儿歌 / AI
- **顶部状态**：显示当前年龄模式、设置入口
- **首页内容**：根据年龄模式推荐对应模块卡片，显示学习进度
- **年龄分级**：使用用户提供的三张真实图片（幼儿园/小学生/初中生）作为身份标识

### 4.2 字母认读
- **交互形式**：左右滑动卡牌切换 26 个字母
- **卡牌内容**：大写 + 小写 + 音标 + 关联单词 + 图片
- **控制按钮**：◀ 上一个 / 🔊 发音 / 🎤 跟读评测 / ▶ 下一个
- **进度指示**：圆点进度条（绿色已学/粉色当前/灰色未学）
- **评测反馈**：音素级评分显示（✅正确/⚠️需注意/❌再练）

### 4.3 音标认读
- **布局**：元音 / 辅音 选项卡分区
- **音标列表**：美式音标，点击进入详情
- **详情页**：大号音标 + 🔊 听发音 + 🎤 跟读（按钮 80px）+ 发音要领 + 3个例词

### 4.4 单词跟读
- **分类入口**：大图标网格（动物/水果/颜色/数字/身体/家庭），2列布局
- **学习页**：180px 大图 + 48px 英文单词 + 音标
- **超大按钮**：🔊 听发音 / 🎤 跟读 / 🔄 重听（各 88px）
- **评测结果**：综合评分 + 音素级逐项反馈
- **切换**：左右滑动切换单词

### 4.5 儿歌播放
- **列表页**：大卡片纵向滑动列表（80px 封面 + 20px 标题）
- **播放器**：
  - 顶部显示歌名 + 封面小图标
  - 歌词滚动高亮（当前行居中放大粉色高亮，已唱淡出，未唱待滚动）
  - 进度条可拖动
  - 控制按钮：⏮ / ⏸ / ⏭

### 4.6 美文阅读
- **分类入口**：大卡片纵向滑动列表（故事/科普/日常/寓言等）
- **阅读页**：
  - 🔊 朗读 / 🎤 跟读（88px 超大按钮）
  - **朗读模式**：TTS 逐句朗读，句子居中放大高亮+自动滚动（类似歌词效果）
  - **跟读模式**：逐句跟读 + 发音评测
  - **点词翻译**：点击单词弹出翻译/发音/音标
  - 字体大小可调
- **300 篇美文**：按三级难度分配，每级约 100 篇

### 4.7 AI 对话（豆包大模型）
- **AI 老师名称**：Elizabeth
- **入口**：Elizabeth 头像 + START 按钮，点击直接开始对话
- **对话界面**：
  - AI 头像使用用户提供的英语老师图片
  - 消息气泡（AI 蓝色 / 用户橙色）
  - 语音输入（🎤）+ 文字输入 + 发送（➤）
  - AI 消息可点击播放发音
- **年龄适配**：
  - 幼儿园：简单短句 + 表情符号 + 慢速
  - 小学：正常语速 + 适量新词
  - 初中：接近母语对话 + 复杂句式

## 5. UI/UX 规范

### 5.1 设计原则
- **大**：按钮 80-120px，图片 120-180px，英文 40-48px
- **简单**：单屏聚焦一个任务，减少文字干扰
- **彩色**：每个模块使用不同配色
- **反馈**：每个操作都有视觉/听觉反馈

### 5.2 字体
- 英文：系统字体，大号突出显示
- 汉字：仅用于辅助说明，字号偏小不喧宾夺主

### 5.3 配色
| 模块 | 主色 |
|------|------|
| 字母认读 | 彩色渐变 |
| 音标认读 | 粉色系 |
| 单词跟读 | 分类各自配色 |
| 儿歌播放 | 暖橙/红 |
| 美文阅读 | 蓝色/绿色 |
| AI 对话 | 蓝色（AI）/ 橙色（用户） |

## 6. 数据与内容

### 6.1 内置内容
- **字母**：26 字母 + 大小写 + 音标 + 关联单词 + 配图
- **音标**：美式音标列表 + 发音要领 + 例词
- **单词库**：按主题分类（动物/水果/颜色/数字/身体/家庭等）
- **儿歌**：经典英文儿歌音频 + 同步歌词
- **美文**：300 篇内置文章，分级存储

### 6.2 用户数据（LocalStorage / IndexedDB）
- 学习进度（已学字母/单词/文章）
- 评测历史
- 当前年龄模式设置
