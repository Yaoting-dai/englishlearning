整体架构设计

第1部分：技术架构 & 路由结构

技术栈总览

前端框架

Next.js 14 (App Router) + TypeScript

样式方案

Tailwind CSS + 响应式布局 (iPad 优先)

离线能力

PWA (Service Worker + Manifest)

AI 对话

火山引擎 ARK (豆包 API)

发音评测

Azure Speech Pronunciation Assessment

语音合成 (TTS)

Azure TTS / Web Speech API

页面路由结构

/ ← 首页（根据年龄推荐内容）
/letters ← 字母认读（A-Z）
/letters/[id] ← 单个字母学习
/phonetics ← 音标认读
/words ← 单词跟读
/words/[category] ← 分类单词（动物/水果/颜色...）
/songs ← 儿歌播放
/reading ← 美文阅读 ← 按难度分级
/reading/[id] ← 单篇文章阅读
/ai-chat ← AI 对话（老师/陪练）
/settings ← 家长设置（年龄/难度/进度）
