音色方案

全应用统一使用女童音色

可选方案

A

Azure AnaNeural（推荐）

Azure 官方提供的 en-US-AnaNeural 是纯正女童音色（"neutral kid voice"）

✅ 原生儿童音色，无需调参

✅ 与发音评测同属 Azure，统一管理

✅ 通过 SSML 调 pitch/rate 可进一步低龄化

B

ElevenLabs 儿童音色

ElevenLabs 有更逼真的儿童音色选项，甚至支持声音克隆

✅ 音质更加自然逼真

⚠️ 需要额外账户和费用

⚠️ 与评测服务分开管理

C

Azure 成人音色 + SSML 调音

用 Jenny/Aria 等成人女声 + 提高音调 + 加快语速

✅ 零额外成本

⚠️ 效果不够自然

⚠️ 长时间听容易疲劳

建议：先用 Azure AnaNeural（方案A），后续如果需要更逼真的效果，可升级到 ElevenLabs。
